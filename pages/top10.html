Les OS

Voici le top 6 des OS que je préfère

1. Debian sans interface (Pour les serveurs c'est toujours le meilleur)

2. Bazzite sous KDE (Pour le gaming et la burautique il est parfait pour les fans de KDE)

3. Linux Mint (Pour une simplicité maximal et une performance parfaite)

4. Garuda Linux (Pour les fans de hyperland, il est optimisé pour.)

5. Fedora (Basé sur red hat, donc très sécurisé et parfait pour le pen testing)

6. Arch (Très ouvert, donc interface que vous voulez facilement.)
